Guidelines for Great Feedback

Bug Reports

  1. Search first: Check if the issue already exists
  2. Be specific: Include macOS version, MGenie version, and exact steps to reproduce
  3. Attach logs: Include console output or crash reports if available
  4. Screenshot/video: Visual aids help us understand the problem faster

Feature Requests

  1. Describe the use case: What problem does this solve?
  2. Explain the benefit: Who would benefit and why?
  3. Keep it focused: One feature per request
  4. Check for duplicates: Search existing requests first

Other Ways to Connect

Email us at feedback@mgenie.app for private feedback or general inquiries.
