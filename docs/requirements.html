System Requirements

Everything you need to run MGenie on your Mac.

Minimum Requirements

🍎

macOS Version

macOS 13 (Ventura) or later

🖥️

Processor

Intel or Apple Silicon (M1, M2, M3, etc.)

💾

Storage

~50 MB of available disk space

🔊

Microphone

Required for transcription and voice activation features

Recommended Requirements

For the best experience, especially with AI features:

⚡

Apple Silicon Mac

M1 or newer for optimal performance and Apple Intelligence support

🧠

Apple Intelligence (Optional)

macOS 26 (Sequoia) with 16GB+ RAM for on-device AI features

🌐

Internet Connection

Required only if using external LLM backend or cloud transcription

Permissions Required

🎤

Microphone Access

Grant access in System Settings → Privacy & Security → Microphone to enable recording and transcription.

🗣️

Speech Recognition

Grant access in System Settings → Privacy & Security → Speech Recognition for live transcription.

⌨️

Accessibility (Optional)

Only needed for global keyboard shortcuts. Grant in System Settings → Privacy & Security → Accessibility.

💡 Tip: MGenie works entirely offline if you don't use the LLM backend. Speech recognition and AI summaries can run on-device without an internet connection.

Browser Compatibility

This website works best in modern browsers (Safari, Chrome, Firefox, Edge). For the best experience on macOS, use Safari.
