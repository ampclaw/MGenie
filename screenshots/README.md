## How to Add Screenshots

1. Take a screenshot of MGenie in action using `⌘ Shift 4`
2. Save it with a descriptive name (e.g., `notch-area.png`)
3. Add it to this folder
4. Update the image references in `/index.html`

## Recommended Screenshot Sizes

- Width: 1200-1400px (for 2x Retina displays)
- Format: PNG or WebP (compressed)
- Aspect ratio: 16:9 or 4:3

## Pro Tips

- Capture in Light mode for clear visibility
- Include UI elements that showcase key features
- Add annotations or arrows to highlight specific areas
- Show before/after states for features like AI summarization
